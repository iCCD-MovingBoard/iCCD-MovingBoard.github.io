# 動く板NEOの特徴

- **屋外走行可能**
- 最大積載重量 **200kg**
- 最高速度 **15km/h**
- 走行可能時間 **8時間** (1回の充電)
- 自動運転レベル4 (<u>限定された場所での自動運転</u>)
